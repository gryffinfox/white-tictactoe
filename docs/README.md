## Project set up
**1. Create virtual environment**

    python3 -m venv name-of-your-environment

**2. Activate virtual environment**

    source ./name-of-your-environment/bin/activate #linux

    venv\Scripts\activate #Windows

**3. Clone git repository**

    git clone https://github.com/gryffinfox/white-tictactoe.git

**4. Navigave into the project directory**

    cd white-tictactoe
    
**5. Install requirements**

    pip install -r requirements.txt

**Run the game locally**
`python3 manage.py runserver` >>> [http://127.0.0.1:8000/](http://127.0.0.1:8000/)
